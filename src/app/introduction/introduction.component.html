CLXBookshelf Application using Angular, ASP.Net Core Web API and .Net Core 2.1

Welcome to User CLXBookshelf single-page application, built with:

  • .Net Core 2.1

  • Node.js

  • Angular 6

  • ASP.Net Core API

  • Bootstrap 3

This application is built with the following structure:

  • The Front end UI is in Angular 6. This front end project is in CLXAssignment repository in git hub

  • Back end is 3 tier architectured. This is CLXTest solution created using VS2017. This is added in CLXBackend.zip file in git

    • BAL : Is the project containing Buisness Logic
    • DAL : Is the project containing Database access Logic
    • Models : Is the project containing entities
    • CLXAPI : The APIs are written using .Net core API. User and Books APIs are available in BooksController and UserController respectively inside Controller folder

  • Database: SQL Server 2016 Express is used. EF Database first approach is used to create this database.
    backup file 'TARA.back' for db is uploaded in git repository. Note: connection string in 'CLXAPI\appsettings.json' needs to be changed to make application Up.

  • Images used in the application are stored inside the assets\images sub folder of CLXWeb folder
