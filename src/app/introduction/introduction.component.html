CLXBookshelf Application using Angular, ASP.Net Core Web API and .Net Core 2.1

Welcome to User CLXBookshelf single-page application, built with:

  • .Net Core 2.1

  • Node.js

  • Angular 6

  • ASP.Net Core API

  • Bootstrap 3

User Management application is built with the following structure:

  • The Front end UI is in Angular 6. This front end project is in CLXWeb folder

  • Back end is 3 tier architectured.

    • BAL : Is the project containing Buisness Logic
    • DAL : Is the project containing Database access Logic
    • Models : Is the project containing entities

  • The APIs are written using the Controllers and are placed inside Controllers folder. User and Books APIs are available in BooksController and UserController respectively

  • The CRUD Operation logic is written in UserDataAccessLayer.cs file in UserDataAccessLayer Folder

  • Database SQL Server 2016 Express is used. EF Database first approach is used to create this database.
    .bak file for db is stored in CLXDb project. Note: connection string in 'CLXAPI\appsettings.json' needs to be changed.

  • Images used in the application are stored inside the assets\images sub folder of CLXWeb folder
